# 界面截图清单

以下截图建议用于软著申报材料、成果展示材料和内部验收材料。

1. 首页：系统名称、用途、数据闭环入口。
2. 登录页：管理员和医生登录入口。
3. 管理员后台：医生账号创建、全量患者列表、系统统计概览。
4. 医生工作台：患者建档表单、患者列表、患者详情。
5. 患者二维码/链接区域：二维码展示、链接复制、链接启用/禁用。
6. 患者随访页：知情同意提示、随访表单。
7. 医生趋势图：睡眠、焦虑、抑郁、心烦评分趋势。
8. 风险预警区域：连续评分升高、漏访、不良事件记录、自行停药提示。
9. 不良事件处理区域：处理状态、医生备注、处理记录。
10. 数据统计页：基线人数、完成率、基线对比图、自动研究摘要。
11. 数据导出页：脱敏确认、CSV/Excel 导出按钮。
12. 系统展示页：数据闭环流程图、核心指标卡、趋势图、虚拟患者样例数据。
