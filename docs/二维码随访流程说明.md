# 二维码随访流程说明

## 一、流程定位

二维码随访流程用于医疗机构制剂柴牡开郁颗粒用药后的真实世界随访、疗效观察、安全性记录和研究数据采集。患者不需要注册账号，也不需要手工输入任何随访识别信息。

## 二、医生端生成流程

1. 医生登录医生工作台。
2. 医生创建或选择本人管理的患者档案。
3. 系统为患者自动生成研究编号和随机随访链接。
4. 医生端患者详情页显示患者专属二维码和随访链接。
5. 医生可复制链接、打印二维码或直接打开患者随访页进行流程核验。
6. 二维码生成、链接启用或禁用等关键操作写入审计日志。

## 三、患者端填写流程

1. 患者使用医生提供的二维码进入本人专属随访页面。
2. 页面展示研究编号、中医诊断、证型、当前随访节点和知情同意提示。
3. 首次填写前患者需勾选确认知情同意提示。
4. 患者按实际情况填写服药、睡眠、情绪、症状、量表、安全性和备注信息。
5. 提交后记录进入医生端和数据统计页。

## 四、权限和安全设计

- 患者链接使用随机识别信息生成，但页面不提供查询或手工输入入口。
- 患者只能通过本人专属链接访问本人随访页。
- 医生可禁用或重新启用患者随访链接。
- 数据库本地保存，手机号可选填并在页面默认脱敏显示。
- 数据统计和脱敏导出默认优先使用 research_id。

## 五、边界说明

二维码随访流程仅用于随访和数据采集，不产生诊断、处方、剂量调整或线上问诊服务。患者如出现明显不适，应及时联系医生或到医疗机构就诊。
