# 量表模块说明

## 模块定位

量表模块用于在患者随访时补充标准化睡眠、焦虑和抑郁信息。为降低患者填写压力，量表区域在患者端默认折叠，患者可按需要展开填写。

## PSQI简化睡眠评估

包含入睡困难、夜间易醒、早醒、睡眠时长、白天疲乏五项，每项 0-3 分，系统自动计算 `psqi_simple_score`，总分范围 0-15。

## GAD-7焦虑量表

包含 7 个条目，每项 0-3 分，系统自动计算 `gad7_score`，总分范围 0-21。

| 总分 | 分级 |
| --- | --- |
| 0-4 | 正常或轻微 |
| 5-9 | 轻度 |
| 10-14 | 中度 |
| 15-21 | 重度 |

## PHQ-9抑郁量表

包含 9 个条目，每项 0-3 分，系统自动计算 `phq9_score`，总分范围 0-27。

| 总分 | 分级 |
| --- | --- |
| 0-4 | 正常或轻微 |
| 5-9 | 轻度 |
| 10-14 | 中度 |
| 15-19 | 中重度 |
| 20-27 | 重度 |

## 展示与导出

医生端展示单个患者量表趋势。数据统计页展示 PSQI 简表、GAD-7、PHQ-9 的 84 天趋势和单个患者轨迹。CSV/Excel 导出包含量表总分和分级。

## 使用边界

量表分数仅用于随访观察和科研数据记录，不作为自动诊断依据，不产生治疗建议。
