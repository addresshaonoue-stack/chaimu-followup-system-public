# 用户使用手册

## 医生使用流程

1. 登录医生账号。
2. 在医生工作台创建患者档案，填写姓名、性别、年龄、中医诊断、证型、开始用药日期等信息。
3. 系统自动生成研究编号、随访计划、随机 token 链接和二维码。
4. 将随访链接或二维码提供给患者。
5. 在患者详情页查看随访完成情况、评分趋势、量表趋势和风险预警。
6. 对不良事件记录进行处理，填写医生备注和处理记录。
7. 在医生综合观察评价模块新增或修改医生综合观察评价和安全性评价。

## 患者使用流程

1. 通过医生提供的专属链接或二维码进入患者随访页。
2. 首次填写前阅读知情同意提示并勾选确认。
3. 选择待填写随访节点。
4. 填写服药情况、睡眠、情绪、主要症状、安全性和备注。
5. 可按需展开量表模块，填写 睡眠自评简表、GAD-7、PHQ-9。
6. 提交后该节点记录进入医生端和数据统计页。

## 科研人员使用流程

1. 登录医生或管理员账号。
2. 进入数据统计页查看总体指标、完成率、基线对比、量表趋势、依从性分布、医生综合观察评价分布和安全性分布。
3. 在患者完整数据区点击单个患者查看个人 84 天趋势和量表轨迹。
4. 使用统计 CSV/Excel 或脱敏 CSV/Excel 按钮导出数据。

## 注意事项

患者端不显示医生内部医生综合观察评价。系统仅用于随访、应用观察、安全性记录和研究数据采集，不提供在线诊疗、生成诊断结论、自动开方或剂量调整。
