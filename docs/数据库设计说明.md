# 数据库设计说明

## 数据库类型

SQLite/sql.js，本地文件默认保存于 `data/chaimu_followup.sqlite`。系统启动时会执行兼容性迁移，旧数据库缺少字段时自动补齐。

## users

医生和管理员账号表。

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| username | 登录账号 |
| password_hash | 密码哈希 |
| role | admin 或 doctor |
| display_name | 显示姓名 |
| department | 科室 |
| is_active | 是否启用 |
| created_at | 创建时间 |

## patients

患者档案表。

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| patient_id | 系统患者编号 |
| research_id | 研究编号，格式如 CMKY-2026-0001 |
| token | 患者随访随机 token |
| token_disabled | token 是否禁用 |
| token_disabled_at | token 禁用时间 |
| name | 姓名 |
| sex | 性别 |
| age | 年龄 |
| phone | 手机号，可选 |
| diagnosis | 中医诊断 |
| tcm_syndrome | 中医证型 |
| medication_start_date | 开始用药日期 |
| doctor_id | 管理医生 |
| consent_signed | 是否签署知情同意 |
| consent_time | 知情同意确认时间 |
| notes | 备注 |

## followup_schedules

随访计划表，保存固定节点和医生自定义节点。

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| patient_id | 患者 |
| label | 节点名称 |
| due_date | 计划随访日期 |
| is_custom | 是否自定义 |
| created_by | 创建医生 |

## followups

随访记录表。

| 类别 | 字段 |
| --- | --- |
| 基本信息 | patient_id、schedule_id、visit_label、visit_date、submitted_at |
| 服药情况 | on_time_medication、missed_doses、self_discontinued |
| 睡眠与情绪 | sleep_score、sleep_difficulty、early_wake、vivid_dreams、sleep_hours、anxiety_score、depression_score、irritability_score、low_mood_score |
| PSQI简表 | psqi_sleep_latency、psqi_night_wake、psqi_early_wake、psqi_sleep_duration、psqi_day_fatigue、psqi_simple_score |
| GAD-7 | gad7_1 至 gad7_7、gad7_score、gad7_level |
| PHQ-9 | phq9_1 至 phq9_9、phq9_score、phq9_level |
| 主要症状 | chest_tightness、palpitation、dizziness、bitter_mouth、dry_mouth、fatigue |
| 依从性 | adherence_forget、adherence_stop_better、adherence_stop_discomfort、adherence_regular、adherence_score、adherence_level |
| 安全性 | has_adverse_reaction、adverse_type、severity、stopped_due_adverse、sought_medical_help、adverse_description |
| 备注与处理 | patient_note、doctor_note、doctor_handling_status、doctor_handling_note、handled_by、handled_at |

## clinician_evaluations

医生疗效评价表。

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| patient_id | 患者 |
| evaluation_date | 评价日期 |
| clinician_effect | 显效、有效、无效、加重、暂未评价 |
| clinician_safety | 未见明显不良反应、轻度、中度、重度不良反应 |
| clinician_note | 评价备注 |
| evaluator_doctor | 评价医生 |
| created_at / updated_at | 创建与更新时间 |

## doctor_notes

医生备注表，记录患者随访管理过程中的内部备注。

## audit_logs

审计日志表。

| 字段 | 说明 |
| --- | --- |
| id | 主键 |
| actor_id | 操作人账号 ID，可为空 |
| actor_name | 操作人名称 |
| actor_role | 操作人角色 |
| action_type | 操作类型 |
| object_type | 操作对象类型 |
| object_id | 操作对象编号 |
| summary | 操作摘要 |
| created_at | 操作时间 |
