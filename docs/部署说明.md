# 部署说明

## 一、运行环境

- Node.js 18 或更高版本。
- Windows、本地电脑、医院内网服务器或云服务器均可部署。

## 二、本地启动

```bash
npm install
npm run init-db
npm start
```

访问地址：

```text
http://localhost:3000
```

## 三、内网服务器部署

1. 将项目复制到医院内网服务器。
2. 执行 `npm install --production`。
3. 首次部署执行 `npm run init-db`。
4. 设置环境变量：

```bash
PORT=3000
SESSION_SECRET=请替换为强随机字符串
DB_PATH=/data/chaimu_followup.sqlite
```

5. 使用进程管理工具运行 `npm start`。
6. 配置数据库文件备份、访问控制和服务器安全策略。

## 四、云服务器部署

云服务器部署应启用 HTTPS，限制后台访问范围，设置强密码和强随机 SESSION_SECRET，并建立安全更新、日志留存和数据库备份机制。

## 五、运行维护

建议配置定期备份、账号权限管理、服务监控和导出文件留存目录，便于小范围试用、成果展示和后续版本迭代。
