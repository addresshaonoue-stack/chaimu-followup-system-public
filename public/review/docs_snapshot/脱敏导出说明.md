# 脱敏导出说明

## 导出入口

系统在数据统计页和数据导出页提供 CSV/Excel 导出。数据导出页默认勾选脱敏导出。

## 脱敏导出规则

脱敏导出时仅保留 `research_id` 作为患者识别编号，不导出姓名和手机号。导出仍保留性别、年龄、诊断、证型、开始用药日期、各随访节点数据、量表总分、量表分级、依从性评分、不良事件记录、医生综合观察评价和医生备注。

## 非脱敏导出

非脱敏导出包含 `patient_id` 和姓名，适用于院内授权范围内的数据核对。系统仍不导出手机号字段。

## 导出审计

每次导出都会写入审计日志，记录导出人、角色、导出类型、导出时间和摘要。

## 使用建议

用于论文统计、成果转化材料、会议展示或跨部门流转时，应优先使用脱敏导出文件，并按所在机构数据管理要求进行保存和传输。
