# 真实世界研究方案

## 一、研究背景

柴牡开郁颗粒作为医疗机构制剂，其临床应用过程需要持续、规范、可追溯的数据记录。真实世界随访平台用于院内制剂应用观察、患者随访数据采集、医生综合观察评价和数据质量管理，为后续真实世界研究、成果转化和规范化评价提供基础。

当前平台样例数据为演示数据，正式真实世界研究需经过伦理审查、知情同意和规范数据管理后方可开展。

## 二、研究目的

1. 建立患者建档、二维码随访、量表采集、医生审核、数据统计和脱敏导出的闭环流程。
2. 观察情志相关症状、睡眠自评、中医症状积分和依从性等指标的变化趋势。
3. 规范记录不良事件、漏访、补访、线下复诊或转介情况。
4. 为院内制剂应用评价、科研课题、论文、软著和成果转化资料提供结构化数据支撑。

## 三、研究设计

拟采用前瞻性、观察性、真实世界随访研究设计。研究不改变医生原有诊疗决策，不提供在线诊疗、不生成诊断结论、不自动开方、不自动调整用药剂量。

## 四、研究对象

在医疗机构内接受柴牡开郁颗粒应用观察并同意参与随访数据采集的患者。

## 五、纳入标准

1. 符合院内制剂应用观察管理要求。
2. 年龄、性别不限，具备完成随访问卷能力。
3. 已完成知情同意。
4. 可通过二维码或专属链接完成随访。

## 六、排除标准

1. 无法完成随访或无法提供有效知情同意。
2. 研究者判断不适合纳入随访观察的数据记录。
3. 存在严重资料缺失，无法完成基线记录。

## 七、退出与脱落标准

1. 患者主动撤回知情同意。
2. 连续多次漏访且无法联系。
3. 医生判断继续随访不适合。
4. 数据合规或伦理要求需停止采集。

## 八、随访时间点

固定随访时间点包括：

- 基线
- 第7天
- 第14天
- 第28天

系统可继续扩展第56天、第84天和医生自定义随访节点，用于更长周期观察。

## 九、主要观察指标

1. 情志相关症状评分变化趋势。
2. 医生综合观察评价：明显改善、部分改善、暂未见明显改善、需进一步评估、暂未评价。

## 十、次要观察指标

- 睡眠自评。
- 中医症状积分。
- 随访完成率。
- 数据完整率。
- 依从性评分与分布。

本睡眠自评简表参考睡眠相关维度设计，仅用于院内随访观察，不等同于标准诊断工具。

## 十一、安全性指标

- 不良事件记录。
- 严重不良事件。
- 线下复诊或转介情况。
- 停药、就医、处理状态和医生备注。

## 十二、PHQ-9第9项阳性处理流程

1. 系统仅提示医生关注，不作诊断结论。
2. 医生应结合线下问诊、病史和必要时心理专科评估进行处理。
3. 必要时进行心理专科转介或紧急处理。
4. 处理过程应记录处理状态、处理备注、处理人和时间。

## 十三、不良事件分级与处置

不良事件类型包括胃肠不适、头晕、皮疹、口干、嗜睡和其他。严重程度分为轻度、中度、重度。与用药相关性由医生判断，可记录为可能相关、可能无关、无法判断或待医生评估。

处理状态包括记录、随访、线下复诊、转介、关闭。

## 十四、漏访提醒流程

系统根据随访计划识别未完成节点，医生端显示逾期随访和需关注患者。医生可联系患者、记录原因、安排补访或关闭该节点。

## 十五、缺失数据处理

1. 记录缺失原因，包括漏访、补访、撤回、数据异常等。
2. 描述性统计中标明分母和有效记录数。
3. 趋势图不对缺失值做未经说明的填补。
4. 正式研究统计分析中需预先设定缺失值处理策略。

## 十六、数据质量控制

包括完整性、及时性、一致性、可追溯性、脱敏合规性和审核闭环率。系统通过审计日志记录建档、随访提交、医生备注、医生综合观察评价、数据导出和token状态变更。

## 十七、数据脱敏和隐私保护

科研展示默认使用研究编号，不显示姓名和手机号。不采集身份证号。手机号可选填，并在医生端脱敏显示。脱敏导出默认不导出姓名和手机号。

## 十八、统计分析计划

1. 描述性统计：建档人数、随访记录数、完成率、数据完整率、不良事件记录数。
2. 前后变化趋势分析：基线、第7天、第14天、第28天等节点的量表和症状评分趋势。
3. 亚组探索：年龄、性别、证候、依从性等分层观察。
4. 不良事件汇总：类型、严重程度、相关性判断和处置状态。
5. 不作未经验证的疗效结论，不使用绝对化宣传表达。

## 十九、伦理与知情同意

正式研究开展前需通过医院伦理审批、信息安全审批和数据合规审查。患者需知情同意后方可纳入数据采集。

## 二十、研究局限性

真实世界数据存在漏访、选择偏倚、合并用药、患者自评误差和缺失数据等局限。平台仅提供数据采集与展示工具，研究结论需结合规范统计分析和伦理批准后的真实数据。

## 二十一、后续真实研究计划

后续可在伦理审批后开展多中心或更长周期随访，完善样本量估算、统计分析方案、数据监查计划和研究报告规范。
