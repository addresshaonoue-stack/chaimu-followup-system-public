# 管理员使用手册

## 登录

访问 `/login.html`，使用管理员账号登录。云端首次启动时通过环境变量 `ADMIN_USERNAME` 和 `ADMIN_PASSWORD` 创建初始管理员；本地演示环境可按 README 中的演示账号进入。正式使用时应配置独立 `SESSION_SECRET` 并妥善保存管理员密码。

## 创建医生账号

进入管理员后台，在“创建医生账号”模块填写用户名、初始密码、医生姓名和科室。创建后医生可登录系统管理本人患者。

## 查看全部患者

管理员后台的“全部患者”表格显示研究编号、patient_id、姓名、脱敏手机号、性别/年龄、诊断、证型、管理医生、随访数、知情同意和链接状态。

## 查看数据统计

管理员可进入数据统计页查看全部患者数据，包括随访完成率、量表趋势、依从性分布、医生综合观察评价分布和安全性评价分布。

## 数据导出

管理员可在数据统计页或数据导出页导出 CSV/Excel。建议优先使用脱敏导出，仅保留 research_id，不导出姓名和手机号。

## 审计日志

管理员后台提供审计日志页面，记录创建患者、提交随访、医生备注、医生综合观察评价、不良事件处理、token 禁用/启用和数据导出等操作。日志用于系统追溯和数据质量核查。

## 合规提醒

正式用于临床前需完成医院伦理审批、信息安全审批和数据合规审查。系统不得作为在线诊疗、生成诊断结论、自动开方、剂量调整或互联网医院服务使用。
